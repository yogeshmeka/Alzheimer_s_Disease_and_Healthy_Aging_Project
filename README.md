# Alzheimer_s_Disease_and_Healthy_Aging_Project
🔍 Objective: To perform early detection of Alzheimer's based on cognitive health indicators.
📊 Data Source: Behavioral Risk Factor Surveillance System (CDC).
🧩 Approach: Data Preprocessing, EDA, and predictive modeling using Machine Learning (Support Vector Machine, K-Nearest Neighbors, Random Forest).
📈 Outcome: Identifying major factors influencing cognitive health and developing a high-performance prediction model.
